Hydrological analysis in R, Python, Matlab, and others

Over the years, we've written quite a bit of software. Any code that we think may be useful for others is hosted on our Github page. Some projects worth mentioning are:

  • Topmodel is an implementation of the hydrological model TOPMODEL as a dynamic library, wrapped into an R package.
  • r_fuse is an R implementation of the FUSE hydrological modelling framework.
  • The Riverlabs project hosts the software for our Riverlabs suite of low-cost water quantity and quality sensors.

Many group members have their own Github page. For example, previous group member Claudia Vitolo has built produced several R packages.
